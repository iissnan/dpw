Le Caraco - Spring in Action 4

发表于 2016-09-01

在尝试结合 Spring 官方文档来直接阅读 DPW 后端源码失败后，我决定挑选几本关于 Spring 的书籍。这其中还包括一些 Java 语言相关的书籍，作为基础支撑，然后通过阅读现有的代码来巩固（或补足）基础知识，如此循环渐进。《Spring in Action》是 Spring 相关书籍中的第二本，在阅读了其第一章后，感觉比较清晰（参照物包括其它书籍以及 Spring 官方文档），因此将此本书作为学习 Spring 的主要资料。

但阅读英文技术书籍的速度还是太慢，于是我购买了中文翻译版。阅读电子版本书籍的唯一好处或者坏处是不能很直观的感觉到书籍的内容量，当收到《Spring in Action》，我才惊讶地发现这本书的厚度与《Head First Design Pattern》的厚度相当。

不过这并不影响到学习的热情，9 月份的目标是至少看过本书的 四分之三。按照每天一个小时的阅读进度应该不难达到，但需要结合动手练习以及结合当前项目的源码，这部分时间目前很能预估。或许在一周之后会比较明朗。

此为开篇。
