How objects are constructed

Several things happen when you use the new operator. The new instance of the given class is created, memory is allocated for it, and a special method defined in the given class is called. This method is called a constructor.

A constructor is a way to create a new instance of a class. A constructor initializes the new object and its variables, creates any other objects that one object needs, and performs any additional operations the object requires to initialize itself. A class can have several different constructors, each with a different number or type of argument. When you use new, you can specify different arguments in the argument list, and the correct constructor for those arguments is called.

No two constructors in a class can have the same number and type of arguments, because this is the only way constructors are differentiated from each other.
If a class defines no constructors, a constructor with no argument is called by default when an object of the class is created. The only thing this constructor does is call the same constructor in its superclass.

Class variables

Class variables are variables defined and stored in the class itself. Their values apply to the class and all its instances. With instance variables, each new instance of the class gets a new copy of the instance variables. With class variables, only one copy of that variable exists when the class is loaded. Changing the value of that variable changes if for all instances of that class.

Calling Methods

Methods of an object are called to make it do something. Calling a method in an object also makes use of dot notation.

Class Methods

Class methods, also called static methods, apply to the class as a whole and not to its instances just like variables. Class methods commonly are used for general utility methods that might not operate directly on an object.

Dot notation is used to call a class method. As with class variables, you can use either an instance of the class or the class itself on the left side of the dot.
